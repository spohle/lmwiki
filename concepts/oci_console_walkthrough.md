The Oracle Cloud Infrastructure (OCI) Console serves as the primary web-based interface for managing OCI resources and services. It provides a comprehensive environment for users to interact with their cloud infrastructure.

### Access and Initial View
To access the OCI Console, users navigate to `cloud.oracle.com` and log in using their tenancy, username, and password. Upon successful login, the [[Dashboard]] is displayed, offering quick-start links and shortcuts for launching various resources. The [[Service Navigation menu]], accessible via the **hamburger icon**, categorizes services into logical sections such as [[Compute]], [[Storage]], [[Networking]], and [[Database]], facilitating easy discovery and access.

### Resource Discovery and Management
The console features a robust [[Search Bar]] that allows users to quickly find specific resources or documentation. For more complex queries, the [[Advanced Resource Query]] tool enables running custom searches, such as identifying all "running instances" within an account.

### Region Management
OCI operates across 41+ geographical data center locations, known as [[Regions]]. Users can easily switch between their subscribed regions using a global menu, or subscribe to new regions as needed, enabling global deployment and [[Disaster Recovery]] strategies.

### Developer and Support Utilities
The OCI Console integrates several tools to enhance productivity and support:
*   **[[Cloud Shell]]**: A browser-based bash shell [[Virtual Machine]] (VM) pre-configured with essential tools like the [[OCI CLI]], Git, Java, and Python, ideal for scripting and command-line operations.
*   **Code Editor**: An in-console environment for editing code, useful for updating service workflows or [[Terraform]] scripts directly within the browser.
*   **Announcements**: A dedicated menu to track new feature launches and scheduled maintenance events.
*   **Help/Live Chat**: Provides direct access to the [[Support Center]] for requests like limit increases, or offers immediate assistance via the Live Chat Assistant.
*   **Health Dashboard**: Offers real-time status updates on service degradation across different OCI regions, crucial for monitoring [[High Availability]] and service health.
