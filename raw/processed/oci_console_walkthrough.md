# OCI Console Walk-through

## Access and Dashboard
* **Login**: Access the console via `cloud.oracle.com` using tenancy, username, and password credentials.
* **Dashboard**: The home page features quick-start links and resource launch shortcuts.
* **Navigation**: The **hamburger icon** opens the Service Navigation menu, categorizing services into sections like Compute, Storage, Networking, and Database.

## Search and Regions
* **Search Bar**: Used to find resources or documentation. For example, typing "Compute" will show relevant instances and links.
* **Advanced Resource Query**: Allows users to run sample or custom queries, such as searching for all "running instances" in an account.
* **Region Management**: Users can switch between subscribed regions via a global menu or subscribe to any of the 41+ available geographical data center locations.

## Developer and Support Tools
* **Cloud Shell**: A browser-based bash shell VM pre-installed with the OCI CLI, Git, Java, and Python.
* **Code Editor**: An in-console environment for editing code and updating service workflows like Terraform scripts.
* **Announcements**: A menu for tracking feature launches and scheduled maintenance.
* **Help/Live Chat**: Access the Support Center for limit increases or use the Live Chat Assistant for immediate questions.
* **Health Dashboard**: Provides real-time status of service degradation across different regions.
