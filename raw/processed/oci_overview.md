# OCI Overview

## Service Categories
OCI offers over 80 services categorized into seven major pillars

### 1. Core Infrastructure
* **Compute**: Includes virtual machines, bare metal servers, containers, managed Kubernetes (OKE), and managed VMware services
* **Storage**: Covers disks (block volumes), file storage, object storage, and archive storage.
* **Networking**: Software-defined private networks designed for high security and performance

### 2. Database Services
* **Oracle Databases**: OCI is the only cloud that runs **Autonomous Databases** for OLTP, OLAP, and JSON
* **Open Source**: Managed versions of MySQL and NoSQL are available
* **Infrastructure**: Databases can run on VMs, bare metal, or Exadata Cloud Service

### 3. Data and AI
* **Managed Services**: Includes Apache Spark (DataFlow), Data Catalog for tracking artifacts, and Data Integration for ETL
* **Data Science**: A platform for building and training machine learning models
* **Streaming**: A managed Apache Kafka service for event streaming

### 4. Governance and Administration
* **Security**: Integrated encryption by default, network protection, and cloud security posture management
* **Identity & Management**: Includes **Compartments** to help manage large and complex environments
* **Observability**: Features like logging, logging analytics, and application performance management

### 5. Developer Services & Analytics
* **Resource Manager**: A managed Terraform service
* **APEX**: A managed low-code development service
* **Analytics Cloud**: A managed service that integrates with third-party solutions for data analysis

### 6. Application Services
* **Serverless**: Includes OCI Functions
* **Connectivity**: API Gateway and Events services for building microservices and event-driven architectures

### 7. Application Suite (SaaS)
* A comprehensive suite for business functions including Finance, HR, Supply Chain, Sales, and Marketing, all running on OCI
